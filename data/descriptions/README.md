# GSUI Action Descriptions

`descriptions.json` is a BG-Wiki-sourced description set covering:

- **773 spells** — converted from `FFXIMissingSpells/libs/spell_info.lua`
- **341 job abilities** — scraped from BG-Wiki's `{{Job Ability}}` template
- **226 weapon skills** — scraped from BG-Wiki's `{{Standard WS}}` template

Shipped here so GSUI (or any other addon) can look up action details
without re-scraping. Currently a passive data file — GSUI does NOT yet
display these descriptions; the file is in place for future tooltip /
info-panel features.

## File format

Top-level dict keyed by **lowercase** action name. Each entry:

```json
"cure iv": {
  "name":        "Cure IV",
  "kind":        "spell",            // "spell" | "ja" | "ws"
  "description": "Restores the target's HP.",
  "type":        "White Magic",
  "target":      "Single",
  "mp_cost":     "88 MP",
  "cast_time":   "2.5 seconds",
  "recast_time": "8 seconds",
  "notes":       [ "...", "..." ]
}
```

Other possible keys: `tp_cost`, `duration`, `range`, `skill`, `level`,
`job`, `element`, `effect`.

## Refreshing the data

This file is generated by tools in the **FFXIMacroManager** repo (it's
the primary consumer; GSUI just mirrors the file). To refresh:

1. Re-scrape from FFXIMissingSpells + BG-Wiki:
   ```sh
   cd /c/Users/Jason/Desktop/My\ FFXI\ Add\ ons/FFXIMacroManager
   python tools/convert_spell_descriptions.py
   python tools/scrape_bgwiki_descriptions.py
   ```
2. Copy the result into this folder:
   ```sh
   cp data/descriptions.json \
      /c/Users/Jason/Desktop/Windower/addons/GSUI/data/descriptions/descriptions.json
   ```

The scraper uses BG-Wiki's MediaWiki API (8 batched requests, ~5 seconds)
and caches responses under `FFXIMacroManager/tools/_cache/`. Delete that
folder before step 1 to force a fresh fetch from the wiki.
